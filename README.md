# 군 장병 자살예방 상담 앱 제작 프로젝트

2023 군 장병 AI·SW 교육 하반기 프로젝트 SW02 팀 프론트엔드 개발 팀 레퍼지토리

---

## 기술 스택
* 빌드 툴 -> bun.js
* 사용 라이브러리 -> React Native, expo-sdk
* 상태관리 -> recoil 
* 데이터 패칭 -> graphql, apollo-client

--- 

#### 실행법
```shell
# 의존성 설치
bun install

# expo 로그인 하여 스마트폰에 있는 expo 와 연동하면 스마트폰에서 접속가능
bunx expo login

# 실행
bun start
```
